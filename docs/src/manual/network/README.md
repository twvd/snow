# Networking

Snow supports networking through an emulated Ethernet adapter or LocalTalk over UDP.

<div class="warning">
Connecting a decades-old (emulated) computer, operating system and other software to the public internet comes with security risks.
Ensure you understand the risks and how to protect your network before proceeding.
</div>
